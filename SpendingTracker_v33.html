🗂 Quản lý dữ liệu

Chọn thao tác bạn muốn thực hiện với dữ liệu hiện tại.

1) Giữ lại dữ liệu gần đây

Xóa các giao dịch cũ, chỉ giữ lại một số tháng gần nhất để file JSON nhẹ hơn.

Ví dụ: nhập 6 nghĩa là giữ 6 tháng gần nhất có giao dịch, xóa các tháng cũ hơn.

2) Save dữ liệu theo khoảng tháng

Xuất file JSON hoặc CSV chỉ chứa giao dịch trong khoảng tháng đã chọn.

3) Xóa tất cả

Xóa toàn bộ giao dịch, khoản định kỳ, ngân sách custom và tag custom. Không thể hoàn tác.

⚠️ Xóa khoản thu/chi định kỳ?

Khoản định kỳ này đã tạo giao dịch trong phần “Tất cả giao dịch”.

📥 Import CSV

Bạn muốn import file CSV này theo cách nào?

1) Thêm vào dữ liệu hiện tại

Giữ nguyên dữ liệu đang có và thêm các giao dịch từ file CSV vào danh sách hiện tại.

2) Xóa data hiện tại, add file mới

Xóa giao dịch hiện tại và khoản định kỳ cũ, sau đó nạp các giao dịch từ file CSV.

⚠️ Xóa giao dịch?

Bạn có thể chọn "Sửa" nếu chỉ muốn chỉnh thông tin. Nếu xóa, giao dịch này sẽ bị xóa khỏi dữ liệu.
